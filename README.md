# 📈 CS50 Finance
*A full-stack financial trading platform built with Python, Flask, and SQL.*

## 🚀 Overview
CS50 Finance is a web application that allows users to "buy" and "sell" stocks using real-time market data. This project was developed as part of Harvard's CS50x course, focusing on web security, database management, and API integration.

## ✨ Key Features
* *Real-time Quotes:* Fetches live stock prices using the IEX Cloud API.
* *Portfolio Management:* Track stocks, shares, and current cash balance in real-time.
* *Transaction History:* A complete log of all buy/sell activities.
* *User Authentication:* Secure login and registration system with password hashing.

## 🛠️ Tech Stack
* *Backend:* Python, Flask
* *Database:* SQL (SQLite)
* *Frontend:* HTML5, CSS3, Bootstrap
* *API:* IEX Cloud API

## ⚙️ Setup
1. Clone the repo: git clone https://github.com/abubakarsabir924-cell/CS50-finance.git
2. Set up your API Key from [IEX Cloud](https://iexcloud.io/).
3. Run the Flask server: flask run
